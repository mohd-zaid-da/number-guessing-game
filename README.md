# 🎮 Number Guessing Game (Tkinter)

A simple GUI-based number guessing game built using Python and Tkinter. The player has to guess a randomly generated number between 1 to 100 within limited attempts and a fixed time limit based on the selected difficulty level. This project focuses on basic game logic, user interaction, and GUI development.

## 🚀 Features

- Random number generation between 1–100  
- Multiple difficulty levels (Easy, Medium, Hard)  
- Countdown timer for each game  
- Limited attempts system  
- High / Low feedback on each guess  
- Input validation to handle invalid entries  
- Simple and interactive graphical user interface using Tkinter  

## 🛠️ Tech Used

Python, Tkinter, random module, time module
